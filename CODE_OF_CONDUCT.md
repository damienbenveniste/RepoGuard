# Code Of Conduct

ScaffoldGuard is a small open source project. Participation should stay focused,
professional, and respectful.

## Expected Behavior

- Be direct without being hostile.
- Assume good intent, but respond to concrete impact.
- Keep technical discussion grounded in evidence, reproducible examples, and
  project scope.
- Respect maintainers' decisions about priorities and fit.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Personal attacks or repeated disruption.
- Publishing private information without permission.
- Spam, trolling, or bad-faith issue and pull request activity.

## Enforcement

The maintainer may edit, hide, or delete comments; close issues or pull
requests; block users; or report abuse to GitHub when needed.
